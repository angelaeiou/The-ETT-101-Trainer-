# Amplitude Modulation 🎛️
### Abstract 
This experiment explored Amplitude Modulation (AM) by varying the modulation index and VDC input, generating the message signal and producing a DSB-FC waveform using the Telecoms-Trainer 101.
### Introduction
Amplitude Modulation varies the amplitude of a carrier signal according to a message signal. In this activity, different modulation conditions were applied to observe their effect on the AM waveform and signal envelope.
### Theoretical Background
AM generates sidebands around the carrier frequency, forming a Double Sideband Full Carrier (DSB-FC) signal. Changing the modulation index and VDC input directly affects the envelope shape and signal quality, which can be observed using an oscilloscope.
